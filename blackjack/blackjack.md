# Blackjack
Simple blackjack which follows some of the rules
- A = 1 or 11, whatever is most beneficial
- 2 - 10 = face value
- J, Q, K = 10

## Known issues
- [x] A is only 1, never 11
- [ ] No double down, split or surrender
- [ ] No card limit
- [ ] Dealer draws only 1 card on first
- [x] No variation to payout
